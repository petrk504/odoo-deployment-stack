## Update Policy

### How to Update OCA Modules

1. **Backup first:**
   ```bash
   cd ~/addons/oca/social
   git stash
   ```

2. **Pull latest changes:**
   ```bash
   git pull origin 18.0
   ```

3. **Test on test environment first**

4. **Document new commit hash here**

5. **Update production only after testing**

### Version Tracking

For each module, track the working commit:
```bash
cd ~/addons/oca/social
git log -1 --oneline
# Output: a1b2c3d4... Latest working commit
```

## Troubleshooting

### Module not showing in Odoo
1. Check addons path in docker-compose.yml
2. Verify directory exists: `ls ~/addons/oca/`
3. Check for __manifest__.py: `find ~/addons/oca -name "__manifest__.py"`
4. Restart Odoo: `sudo docker-compose restart`

### Git pull fails
```bash
cd ~/addons/oca/social
git fetch --all
git reset --hard origin/18.0
```

### Need to rollback
```bash
cd ~/addons/oca/social
git reflog  # Find old commit
git checkout <old-commit-hash>
```
